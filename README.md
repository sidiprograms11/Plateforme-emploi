# Plateforme Emploi – Coffre numérique sécurisé

Plateforme de gestion de CV sécurisée pour le gouvernement malien.

## Technologies

Frontend:
- Angular
- Tailwind
- TypeScript

Backend:
- Spring Boot
- Spring Security
- JWT

Base de données:
- PostgreSQL

Cryptographie:
- RSA
- AES
- Signature numérique
- Hash SHA-256

## Lancer le projet

### Backend

```bash
cd demo-backend
./mvnw spring-boot:run
Frontend
cd frontend
npm install
ng serve
